ESK32-370 – HT32F1653/1654 Development Board

The ESK32-370 Development Board is designed for the HT32F1653/1654 series and includes an actual HT32F1654 device. The accompanying peripheral components on the board include a Buzzer, input Keys, RS232 socket, SD Card socket, EEPROM, Serial Flash, LEDs, Potentiometer and Audio in/out etc. This wide range of peripheral components will allow users to quickly understand and evaluate the HT32F1653/1654 functions and features. The ESK32-370 can be used with the e-Link32 Pro debug adapter, providing the user with a complete development environment.
