ESK32-220 – HT32F2755 Development Board

The ESK32-220 Development Board is designed for the CMOS sensor and includes an actual HT32F2755 device. The accompanying peripheral components on the board include a Buzzer, input Keys, Joystick, CMOS sensor connector, RS232 socket, RS485 socket, SD Card socket, Smart Card socket, EEPROM, Serial Flash, LEDs, Potentiometer as well as a 2.8" 320x240 non-touch TFT-LCD module etc. This wide range of peripheral components will allow users to quickly understand and evaluate the HT32F2755 functions and features. The ESK32-220 can be used with the e-Link32 debug adapter, providing the user with a complete development environment.
