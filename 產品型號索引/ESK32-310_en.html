ESK32-310 – HT32F1655/1656 Development Board

The ESK32-310 Development Board is designed for the HT32F1655/1656 series and includes an actual HT32F1656 device. The accompanying peripheral components on the board include a Buzzer, input Keys, Joystick, RS232 socket, SD Card socket, EEPROM, Serial Flash, LEDs, Potentiometer, RJ45 Ethernet module and Audio in/out etc. This wide range of peripheral components will allow users to quickly understand and evaluate the HT32F1655/1656 functions and features. The ESK32-310 can be used with the e-Link32 Pro debug adapter, providing the user with a complete development environment.
