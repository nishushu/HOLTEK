Holtek has released a low speed, long distance data transmission MCU with a wide range 9V~42V operating voltage and an integrated voltage regulator. This MCU can be used in water meters, gas meters and other similar products. Holtek provides a practical reference solution to help users to develop such products rapidly.

Water meters or other similar products, which often need remote automatic reading, are usually several kilometers away from the host and have no power supply. In this case, the RS485 bus that has power and supports remote communication is used in such applications. Due to the expensive RS485 cables and numerous meters, a solution with lower costs and a more convenient wiring scheme is required. The optical direct reading type is still the most popular type among numerous types of water meters. Refer to “Optical Direct Reading Remote Water Meter Solution”.
Additionally, Holtek also provides an emulation host for users to conveniently build a complete system for evaluation.

The associated technical documents can be obtained when linking to the left side diagram. Holtek also provides users with all required technical support related to the emulation host, including circuits, host software and PCBs, etc.

System Architecture and Connection Diagram

In water flow-driven mechanical water meters, an IR LED transmitter and receiver are usually embedded into the disk structure to implement a remote reading function. Using general non-isolated cables combined with a voltage above 26V and a bus parallel mode like MBUS, the system online distance can be up to 1 kilometer or more and more than 250 meters can be connected.

Based on the above market requirements, Holtek provides the HT45F2002 which includes an IR LED transmitter and receiver, a voltage regulator and a high voltage bus transmission functions. To implement a 2-wire bus system for power supply and carrier transmission, each water meter’s average power consumption in this solution can be controlled at 1mA.

This solution also includes an emulation host for development evaluation. Its communication protocol and IR LED optical direct reading principles described in this page can be downloaded. Move the mouse to view the images or download the items required.
