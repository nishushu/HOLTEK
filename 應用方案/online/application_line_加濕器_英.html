Humidifiers are able to refresh air and relieve dryness in a room as well as remove floating particles from the air. They are widely used to regulate the humidity of household or commercial space areas and thus create a comfortable and healthy environment.

Holtek has developed Ultrasonic Humidifier Flash MCUs, the HT45F3820 and the HT45F3830. When compared with traditional MCU solutions for humidifier applications, this device series has an internal diaphragm vibrating frequency control module, water depletion detection function and provides large I/O output current which can directly drive external power transistors. In addition, they use a new high-power nebuliser application design which allows for simultaneous implementation of higher operation efficiencies with no heat dissipation issues.
Holtek provides circuits, application block diagram, PCBs and a component list in this reference design. More information about the competitive advantages of this humidifier solution can be obtained by clicking on the “Learn more about our application solutions”.
