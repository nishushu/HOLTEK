With the ever increasing use of mobile devices, the requirement for long-term usage is increasing the demand for mobile power banks. For these products, the features of multi-purpose applications and simplified volume production have become the product’s development key points.

Holtek has developed a range of MCUs dedicated for use in power bank applications. These MCUs include features such as integrated charge buck PWM control, discharge boost PWM control, power indication management as well as other necessary features. These functions accompanied by an external MOSFET and by examining the power bank product specification requirements, can allow the proper power management device to be selected. The provided power bank application solution has utilised the HT45F4MA as an example to show a reference design for a single-cell dual-2A output power bank. The solution can fully support battery charge and discharge functions and can also automatically detect the connection of an external charge power source.

This reference design provides application circuits, application block diagrams, PCBs and component list as well as other relevant information. For more details regarding the competitive advantages of the Single-cell dual-2A output power bank solution, click on “learn more about our solutions”.

System Architecture and Connection Diagram

This single-cell dual-2A output power bank reference design is developed using the HT45F4MA. Users can modify the charge and discharge software parameters according to their product requirements to change the drive current of the external PMOS and NMOS transistors. A synchronous rectification function will be implemented for the battery during the charging and discharging process, thus minimising power consumption.
