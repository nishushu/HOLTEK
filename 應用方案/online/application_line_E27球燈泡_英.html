Holtek has released a superior E27 LED bulb solution. A 500V high voltage power MOSFET is integrated into the IC reducing the requirement for this external power component. Several protection functions are also included such as VCC Under Voltage Lockout (UVLO), Over Current Protection (OCP), Output LED string open and short protection and VCC Over Voltage Protection (OVP), etc.

To help users to develop the related products quickly, Holtek provides a block diagram, application circuits, PCB, component list and other technical information.

System Architecture and Connection Diagram

This solution uses the HT7L4813 which is a non-isolation buck PWM controller for LED lighting applications. It has a fully integrated PFC circuit which can achieve high power factor values.
In addition, the HT7L4813 includes an integrated 500V high voltage power MOSFET reducing the requirement for this external power component under appropriate power demands.

The HT7L4813 provides several protections such as VCC Under Voltage Lockout (UVLO), Over Current Protection (OCP), Output LED string open and short protection and VCC Over Voltage Protection (OVP), etc.
